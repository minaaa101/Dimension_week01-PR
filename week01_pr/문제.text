2.브랜치를 사용하는 이유는?
기존의 메인 코드에 영향을 주지 않고 새로운 기능 개발이나 수정 작업을 안전하게 수행하기 위해 사용
3.git push는 어디로 보내는걸까?
GitHub와 같은 원격 저장소의 특정 브랜치로 업로드된다.
4.clone과 frok의 차이점은?
